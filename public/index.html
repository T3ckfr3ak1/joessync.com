Four sync modes

Pick the behavior that fits your workflow. Switch at any time.

One-Way Backup

Copy newer and missing files to the destination. The destination is never trimmed — safe for incremental backups.

One-Way Mirror

Make the destination an exact mirror of the source. Optionally remove orphaned files so both sides stay identical.

Two-Way Merge

Sync changes in both directions by last-write time. Conflicts are handled by overwrite or safe move to a trash folder.

One-Way Move

Copy files, verify integrity, then delete the originals. Resume-safe so interrupted moves don't lose data.

Multiple Profiles

Create up to 20 named profiles, each with its own source, destination, mode, filters, and automation settings. Switch between jobs in one click.

Scheduling & Folder Watch

Run syncs on a daily time schedule or trigger automatically when files change. Live folder watch includes configurable quiet periods and byte thresholds.

Filters & Exclusions

Exclude files and folders by pattern. Add an allowlist to sync only what matters. Optional temp-file heuristics keep partial downloads out of your backup.

SHA-256 Manifests

Optionally generate per-folder SHA-256 hash manifests after every sync. Verify your backup is byte-for-byte correct at any time.

Performance Controls

Dial in concurrent file copies (1–32), buffer size, and bandwidth cap in MiB/s. Run at below-normal thread priority to stay out of your way.

Analyze Before You Sync

Run Analyze to preview exactly which files would be copied, moved, or deleted — before a single byte changes. Stop after any file for granular control.
