Four sync modes

Pick the mode that fits what you're doing. You can switch anytime.

One-Way Backup

Copies newer and missing files to the destination without touching anything already there. Good for incremental backups.

One-Way Mirror

Makes the destination an exact copy of the source. Optionally removes files on the destination that no longer exist in the source.

Two-Way Merge

Keeps both sides up to date based on last write time. When there's a conflict, you choose whether to overwrite or move the older file to a trash folder.

One-Way Move

Copies files, verifies them, then deletes the originals. If the sync is interrupted, it picks up safely where it left off.

Multiple Profiles

Set up to 20 profiles, each with its own folders, sync mode, filters, and schedule. Switch between them from the sidebar.

Scheduling & Folder Watch

Schedule syncs at set times, or have the app watch a folder and sync whenever files change. The watch includes a quiet period so it doesn't fire on every single save.

Filters & Exclusions

Skip files and folders by name pattern, or use an allowlist to sync only specific things. There's also an option to ignore temp files so partial downloads don't end up in your backup.

SHA-256 Manifests

After syncing, the app can write SHA-256 manifests for every folder so you can verify your backup is intact at any point.

Performance Controls

Set how many files copy at once (up to 32), adjust buffer size, cap bandwidth usage, and run at low thread priority so it doesn't get in the way.

Analyze Before You Sync

Run Analyze first to see exactly what will happen before anything moves. You can also stop the sync after any individual file.
